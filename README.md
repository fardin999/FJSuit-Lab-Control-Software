**FJSuit** (Fardin Jahanbaksh Shojae Suite) is a proprietary laboratory control and data-acquisition software developed for thermo-fluid and synthetic jet research environments.

### Motion & Actuation
- Controls solenoid valves for synthetic jet and flow actuation
- CNC-style motion control for multi-axis traverse systems
- Optimized smooth-path motion algorithms for accurate positioning
- Compatible with Oriental Motor drivers and stepper systems

### Sensor & Data Acquisition
- Reads environmental data from **Bosch BME280** (Temp/Pressure/Humidity)
- Reads barometric pressure from **Bosch BMP620**
- Remote accelerometer monitoring (ESP32-connected modules)
  - Alerts lab members if equipment loses alignment or stability

### System Integration
- Built in **Python**
- Communications via Arduino / ESP32 microcontrollers
- Real-time logging and system feedback
- Supports expandable sensor and actuator modules
